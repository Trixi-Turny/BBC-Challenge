This project is a web service that makes GET requests to a list of URLs entered by the user and returns the response information usually sent back in the header.


HOW TO RUN:

To Run the program unzip the project folder.
Open project in Eclipse or a similar editor
You can run the Connection or the ConnectionTest classes
If you run Connection you can enter your own urls. Once you are done just press enter twice to finish and the program should display the responses and summary in a json string.



SHORTCOMINGS

This project currently only runs in an IDE, not on the terminal as the classpath could not be configured correctly.
